# Voxlate

[English](#english) | [Русский](#русский)

---

## English

**Voxlate** is a smart and lightweight automation tool designed for seamless voice-to-text translation and text processing.

### Key Features
* **Voice-to-Text Transcription:** Converts speech into clean, formatted text.
* **Smart Translation:** Integrated translation logic for multilingual support.
* **Seamless Automation:** Operates efficiently from a single executable or script, minimizing overhead.
* **Secure Configuration:** Safely stores all local application logs, settings, and API keys in the system's `AppData\Roaming` directory, keeping the source code entirely clean and private.

### Requirements & Setup
The application runs as a standalone Python script or a compiled `.exe` file. On the first launch, it will prompt you to enter your required API keys and will automatically generate the configuration files locally.

## Русский

**Voxlate** — это умный и легковесный инструмент автоматизации, созданный для быстрого перевода голоса в текст и последующей обработки полученного текста.

### Основные возможности
* **Преобразование голоса в текст:** Точная транскрипция речи в чистый, отформатированный текст.
* **Умный перевод:** Встроенная логика для перевода между языками.
* **Полная автоматизация:** Работает в виде одного файла (скрипта или `.exe`), не требуя сложной установки.
* **Безопасная конфигурация:** Программа автоматически сохраняет все рабочие логи, настройки и API-ключи в системной папке `AppData\Roaming`. Исходный код на GitHub остаётся абсолютно чистым и безопасным.

### Запуск и настройка
Программа может запускаться как Python-скрипт или готовый `.exe` файл. При первом старте Voxlate сам предложит ввести необходимые API-ключи и автоматически создаст нужные конфигурационные файлы у вас на компьютере.

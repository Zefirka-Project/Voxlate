Создание репозитория в организации GitHub через Termux (Android)

1. Установить Termux с F-Droid (не из Play Store):
   https://f-droid.org/packages/com.termux/

2. Установить gh CLI:
   pkg install gh

3. Авторизоваться в GitHub:
   gh auth login

4. Создать репозиторий в организации:
   gh repo create zefirka-project/название --private
